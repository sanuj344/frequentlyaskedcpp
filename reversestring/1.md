# 📘 Reverse String – Complete Interview Guide

## 🔹 Problem Statement

Given a string `s`, return the reversed string.

Example:
Input: "hello"
Output: "olleh"

---

# 🔹 Why This Question Matters

Even though this is easy, it tests:

* String traversal
* Index handling
* Two pointer technique
* In-place modification

It builds foundation for:

* Valid Palindrome
* Reverse Words in a String
* Palindromic Substring problems

---

# 🔹 Approach 1: Brute Force

## 💡 Idea

Create a new string and append characters from the end.

## 🧠 Thinking Process (Interview)

"If I traverse from back and build a new string, I can easily reverse it."

## ⏱ Complexity

* Time: O(n)
* Space: O(n)

---

# 🔹 Approach 2: Optimal (Two Pointers)

## 💡 Idea

Use two pointers:

* One at start
* One at end
* Swap characters
* Move inward

## 🧠 Thinking Process (Interview)

"Since strings are mutable in C++, I can reverse it in-place using swapping."

## ⏱ Complexity

* Time: O(n)
* Space: O(1)

---

# 🔹 Interview Tip

If interviewer asks:

👉 "Can you optimize space?"

Immediately say:
"Yes, we can use two pointers and swap in-place to achieve O(1) space complexity."
